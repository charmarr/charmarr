# Setup

Get Charmarr running on your Kubernetes cluster.

## Deployment Options

| Method | Best For |
|--------|----------|
| [Quick Deploy](quickdeploy.md) | Most users - pre-built Terraform blueprints |
| [Manual](manual.md) | Custom setups - Juju CLI deployment |

## Steps

1. **[Prerequisites](prerequisites.md)** - Prepare your environment
2. **Deploy** - Choose Quick Deploy or Manual
3. **[Post-Deployment](post-deploy.md)** - Configure UIs (Plex, Overseerr, etc.)
